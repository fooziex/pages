Linux containers

Rootnode is trying to merge the best parts of shared hosting and VPS. We are proud to introduce Linux containers (LXC), a secure and stable platform to fulfill your deepest desires. :)

A container is a preconfigured, fully-fledged Linux system with great user isolation, guranteed resources and unrestricted shell access to run any software you want.

Web applications

A container is an ideal native environment for hosting web applications in Ruby, Python and PHP. It runs the Apache2 web server with Nginx as a reverse proxy in front, giving you unlimited possibilities for configuration.

We are working on a move to Fastweb, a dedicated, high-availability web app platform, able to process millions of request in the blink of an eye.

No limits

Our users get Rootnode account so they can do things quickly and easily. Because we don't want to make your work harder, we have a very flexible approach to limits.

That means no transfer limits, expansible drive space, no domain limits -- as long as you behave rationally. If you need more resources, ask, and very likely you will get more.

Satan inside

Rootnode users do it differently. Therefore, your account is equipped with Satan, the best command line service manager, developed by us.

With fancy syntax and fast execution, it allows you to configure Rootnode services such as domains, virtual hosts, mailboxes, ftp accounts and carry out many other tasks.
